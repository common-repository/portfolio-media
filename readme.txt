== Description ==

The Portfolio Media plugin allows you to implement a portfolio based gallery. This uses a portfolio media post type to manage portfolio based posts and allows users to attach media objects to them in a similar way 'featured image' works. This works for image and video file formats only.

== Installation ==

1. Upload `portfolio_media.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in Wordpress
3. Create a Wordpress page and implement the [portfolio_media] shortcode.

== Frequently Asked Questions ==

= Can I have multiple different portfolio galleries =

This is not supported
